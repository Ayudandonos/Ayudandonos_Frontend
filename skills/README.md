# Skills del proyecto — Frontend

Skills de Cursor para flujos repetibles. Ubicacion: `.cursor/skills/`

| Skill | Cuando usar |
| ----- | ----------- |
| [create-feature](../.cursor/skills/create-feature/SKILL.md) | Crear feature en `src/features/` |
| [create-page](../.cursor/skills/create-page/SKILL.md) | Crear pagina y registrar ruta |

## Como invocar

En Cursor, referenciar el skill por nombre o pedir al agente que lo lea antes de implementar.

Ejemplo: "Usa el skill create-feature para implementar donations"
